# eCarnet Website

The official introductory website for **eCarnet**, a digital student ID app for all students in Moldova.

## About

eCarnet replaces the traditional paper student ID with a mobile app. It serves as digital proof of student status for public transport, discounts, and institutional access, while also integrating an electronic gradebook, schedule, and personal agenda, and many more features.

## Tech Stack

- HTML, CSS, JavaScript
- Google Fonts (Montserrat + Poppins)
- Mobile-first responsive design
- CSS custom properties for theming
- IntersectionObserver for scroll animations

## Project Structure

```
index.html          — Main page
css/style.css       — Core styles and responsive layout
css/animations.css  — Keyframes and scroll-reveal classes
js/main.js          — Navigation, scroll state, reveal system
assets/             — App icon, SVG icons, images
```

## Running Locally

Open `index.html` in a browser. No build step required.
